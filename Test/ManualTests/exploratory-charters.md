# Exploratory Test Charters

## Charter 1: Security Explanation Quality

Explore whether dashboard messages explain security issues clearly enough for SMB users. Focus on confusing wording, missing evidence, and unclear recommendations.

## Charter 2: External Provider Failure

Explore behavior when one or more external providers are slow, unavailable, or return incomplete data.

## Charter 3: Domain Input Robustness

Explore unusual domain input such as uppercase characters, whitespace, long subdomains, IDN domains, and invalid punctuation.
